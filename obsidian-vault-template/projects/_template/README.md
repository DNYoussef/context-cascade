# Project Template

Copy this folder for each new project.

## Structure

```
project-name/
  README.md       - This file
  context.md      - Project context
  issues.md       - Known issues
  decisions.md    - Architecture decisions
```

## Usage

1. Copy this folder: `cp -r _template my-project`
2. Update context.md with project details
3. Track issues and decisions as you work
